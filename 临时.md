彬，我仔细复核了这版 `seismoelectric_offset_liu2018_spectral.py`。结论先说清楚：

**还不能说“严格按照 Liu et al., 2018 + Schakel & Smeulders, 2010 公式”完成了震电响应模拟。**
它的主框架已经非常接近：确实用了 Schakel 的 (R_E/T_{TM}) 六未知量边界矩阵，也确实做了 Liu 式频率—波数积分；但里面有 **1 个硬公式错误** 和 **1 个方法层面的混合近似问题**，所以目前只能算“接近论文方法的混合实现”，不是严格复现。

---

## 1. 最大硬错误：(\omega_t) 少乘了孔隙率 (\phi)

Schakel 附录 A 里动态渗透率的过渡频率是：

[
\omega_t=\frac{\phi\eta}{\alpha_\infty k_0\rho_f}
]

论文附录 A 明确给出 (\omega_t) 前面有 (\phi)，并且 (M=8\alpha_\infty k_0/(\phi\Lambda^2))。

但代码里现在写的是：

```python
omega_t = eta / (alpha_inf * k0 * rho_f)
```

也就是少了 `phi`。代码这部分确实是在 `dynamic_coefficients()` 里实现 Schakel Appendix A。

应该改成：

```python
omega_t = phi * eta / (alpha_inf * k0 * rho_f)
```

这个不是小问题。因为 (k(\omega)) 和 (L(\omega)) 都直接依赖 (\omega/\omega_t)。你当前 Excel 初始孔隙率大约 (\phi=0.545)，少乘 (\phi) 会让 (\omega_t) 偏大约 (1/0.545\approx1.83) 倍；后期代码把 (\phi) 截到 0.95 时偏差也还有约 5%。这会直接影响动态渗透率、电动耦合系数、有效介电常数，最后影响 (R_E)、(T_{TM}) 和波形幅值。

**这个必须先改。**

---

## 2. Schakel 6×6 边界值求解：主体基本对

这一部分整体是比较靠谱的。

Schakel 论文做的是完整流体—多孔介质界面边界值问题：入射流体 P 波转换为流体侧反射 EM 波、多孔介质侧 TM-EM 波、快慢 P 波和 SV 波；(R_E)、(T_{TM}) 是通过 6 个边界条件组成的线性系统求得，而不是经验拟合。

代码里的未知量顺序是：

[
[R_E,R_M,T_{Pf},T_{Ps},T_{TM},T_{SV}]
]

并建立了 Appendix B 的矩阵元素，包括 B2–B7，对应 Schakel 式 45–46 和附录 B。Schakel 明确给出这些系数定义和六未知量线性系统。 代码中的矩阵搭建部分也确实在按这些项写。

有效密度、慢度根、(\beta)、(\alpha) 幅值比的主线也基本对应 Schakel 式 24–39。论文中 (\beta_m,\beta_n,\alpha_m,\alpha_n) 正是把机械势函数和电场势函数联系起来的幅值比。 代码中这部分对应 `wave_slownesses()` 的慢度根和幅值比计算。

所以：**Schakel 的主边界矩阵部分不是主要问题，主要问题是上游的 (\omega_t) 写错了。**

---

## 3. 动态电化学系数：大体对，但有模型假设

代码里 (C)、pH、(\zeta)、Debye 长度、(L(\omega))、(\sigma_f)、(C_{em})、(C_{os})、(\sigma(\omega))、(\bar{\varepsilon}) 的结构基本对应 Schakel 附录 A。Schakel 附录 A 给出了 (\zeta)、动态电导率、(C_{em})、(C_{os})、Debye 长度等表达式。 代码中也按这个顺序计算。 

但这里有一个需要写进论文方法里的假设：如果没有独立的电解质浓度和流体电导率，代码用

[
C=C_{\rm bg}+[H^+]
]

来估计电解质浓度。这不是 Schakel 原公式，而是你为了把反应输运输出接入震电模型做的近似。代码注释里也承认这是模型假设。

这个可以接受，但论文里不能写成“完全按 Schakel 参数计算”，要写成：

> When only outlet hydrogen concentration was available, the electrolyte concentration was approximated as the background electrolyte concentration plus the outlet hydrogen concentration.

## 4. Liu 2018 频率—波数积分：形式接近，但现在不是纯 Liu Eq. (1)–(2)

Liu 2018 的理论波形是：

[
u_r=\iint A(k,\omega)R_E(k,\omega)
e^{i(k_z^i z_s-k_z^{Er}z+kx-\omega t)}dkd\omega
]

[
u_t=\iint A(k,\omega)T_E(k,\omega)
e^{i(k_z^i z_s+k_z^{Et}z+kx-\omega t)}dkd\omega
]

并且 (R_E)、(T_E) 由 Schakel & Smeulders 2010 计算。刘老师论文里就是这个频率—波数积分框架。

代码的 `synthesize_waveforms_spectral()` 确实按这个形式做了正频率和水平波数求和，并调用 `se_coefficients(..., kx_override=kx)` 逐个 ((k,\omega)) 计算 (R_E/T_{TM})。这一点是对的。 

但是，代码最后又做了一步：

```python
U_real *= finite_offset_dipole_geometry(z_receivers, cfg)[:, None]
```

也就是在已经做完 Liu 式频率—波数积分之后，又额外乘了一个偶极子几何因子。代码自己也说明 `finite_offset_dipole_geometry()` 是 (u\propto \cos\theta/r^2)。

这里就不严格了。刘老师论文中，偶极子模型主要用于解释和验证 Fig. 2(a) 的振幅分布，Fig. 2(b) 是把完整建模结果和偶极子模型做归一化对比；不是在 Eq. (1)–(2) 的频率—波数积分结果上再乘一次偶极子因子。论文和 PPT 都说非零偏移距下振幅先增大后减小，并且可由垂直电偶极子近似解释。

所以现在代码的波形部分是：

[
\text{Schakel }R_E/T_{TM}
+
\text{Liu 频率—波数积分}
+
\text{额外偶极子几何加权}
]

这更像一个“Liu 谱积分 + 偶极子振幅校正”的混合模型。**如果你要严格复现 Liu 2018 Fig. 2(a)，这一步应该去掉；偶极子模型应该作为单独的 Fig. 2(b) 对比曲线，而不是乘回主波形。**

## 5. reduced 模式不是严格正演，只能作为示意图

代码里还有 `synthesize_waveforms_reduced()`，它是：

[
\text{Ricker wavelet} \times |R_E| \times \frac{z}{(D^2+z^2)^{3/2}}
]

这不是 Liu Eq. (1)–(2) 的完整频率—波数正演，只是电偶极子近似波形示意。代码注释也明确说它是 “Ricker wavelet + vertical electric dipole scaling”。

所以以后结果图里要区分：

* `waveform-mode=spectral`：接近 Liu 频率—波数积分；
* `waveform-mode=reduced`：只能叫 dipole approximation / illustrative reduced model。

## 6. 时间因果性处理比之前好，但仍不能用“硬截断”证明

代码现在用了 `causal_ricker_source_spectrum()`，从一个从 (\tau=0) 开始的有限时长 Ricker 源函数计算 (S(\omega))，而不是最后强行把 (T_0) 前面的信号剪掉。这个方向是对的。

但要注意：有限频带频率积分本身会有一点前振铃，这是频谱截断和相位约定造成的数值现象。论文物理上界面 EM 波的主要到时应对应声波到达界面的时间 (T_0=z_s/V_f)，PPT 实验也强调不同电极位置的 SE 波到时基本恒定。

所以你后面判断因果性时，不要只看是否 (T_0) 前完全为 0，而要看 (T_0) 前幅值是否只是很小的数值振铃，主能量是否在 (T_0) 之后。

## 7. 我给这版代码的审核结论

按严格程度打分：

| 模块               | 是否严格对应论文 | 结论                                  |
| ---------------- | -------- | ----------------------------------- |
| Schakel 6×6 边界矩阵 | 基本严格     | 矩阵结构、未知量、(R_E/T_{TM}) 求解主线是对的       |
| Schakel 动态系数     | 部分不严格    | (\omega_t) 少了 (\phi)，必须修            |
| Biot 有效密度与慢度     | 基本严格     | (\rho_{12})、(E_K)、慢度、幅值比主线基本对       |
| Liu 频率—波数积分      | 形式接近     | 核心积分框架有了                            |
| Liu 波形幅值空间分布     | 不严格      | 频率—波数积分后又乘了偶极子因子，属于混合模型             |
| reduced 波形       | 不严格      | 只能作为偶极子近似示意                         |
| 反应输运参数接入         | 可用但有假设   | (H^+\rightarrow C,pH,\zeta) 需要明确是近似 |

**最终判断：这版代码还不能写成“严格按照 Liu et al., 2018 和 Schakel & Smeulders, 2010 完整复现”。更准确的说法是：**

> 代码主体采用 Schakel & Smeulders 的流体—多孔介质界面 6×6 边界值问题计算 (R_E) 和 (T_{TM})，并采用 Liu et al. 的频率—波数积分形式合成 VSEP 界面电磁波形；但当前版本仍存在 (\omega_t) 公式漏乘孔隙率的问题，且波形合成中额外引入了垂直电偶极子几何加权，因此目前属于“谱积分与偶极子近似混合模型”，不是严格的 Liu 2018 全谱积分复现。

## 8. 你现在最该改的两处

第一处，把：

```python
omega_t = eta / (alpha_inf * k0 * rho_f)
```

改成：

```python
omega_t = phi * eta / (alpha_inf * k0 * rho_f)
```

第二处，如果目标是严格 Liu 2018 主波形，把：

```python
U_real *= finite_offset_dipole_geometry(z_receivers, cfg)[:, None]
```

去掉。偶极子模型单独画一张归一化振幅对比图就行，不要乘回谱积分波形。

这样改完后，这版代码才可以比较有底气地说：**Schakel 系数 + Liu 频率—波数积分主线基本严格对应论文。**
