# uploaded_files 文献与材料分类

本目录按当前研究任务整理为三类基础材料 + 一个深度文献讨论笔记目录：

## 01_classical_theory_foundational_papers

经典震电理论、公式来源和奠基性模型论文。优先用于核对 Pride/Schakel/Liu 类公式、边界条件、动态电动参数和频率-波数正演框架。

## 02_recent_seismoelectric_research_experiments_simulations

近年及核心震电研究现状，尤其是实验、数值模拟、谱比、有限偏移距、岩样/砂箱/水槽等可用于引言和讨论的论文。

## 03_previous_reactive_transport_research

此前反应输运、孔隙尺度溶蚀、NMR/反应输运旧稿和补充材料。主要用于把反应输运输出变量与当前震电模型衔接。

## literature_discussion_notes

专门存放“可以与当前论文深入结合讨论的文献笔记”。这里放的是阅读论文后的二次分析，不是普通 PDF 归档。每篇笔记应尽量写清楚：这篇文献能支撑当前论文什么观点、可以放在引言/方法/结果/讨论哪里、能启发哪些新图或敏感性实验、后续还需要补什么。

> 整理原则：PDF 文献按前三类归档；深度阅读、论文写作启发和讨论分析放入 `literature_discussion_notes/`。
