# Literature Discussion Notes

本目录专门用于存放“可以与当前论文深入结合讨论的文献笔记”。

这里不只是放 PDF，也不是普通文献清单，而是用于后续写论文时直接调用的深度阅读记录。每篇笔记建议回答：

1. 这篇论文的 DOI 和基本信息是什么？
2. 它和当前反应输运—震电耦合论文有什么关系？
3. 引用它时可以支撑引言、方法、结果还是讨论中的哪个观点？
4. 它能启发当前论文新增哪些图、对比分析或敏感性实验？
5. 它提醒当前论文哪些机制不能忽略，例如电导率、表面电导、渗透率、频率响应、偏移距效应等？
6. 后续还需要补什么数据、截图、公式核对或代码实验？

## 命名建议

建议使用：

```text
YYYY_FirstAuthor_keyword.md
```

例如：

```text
2006_Block_Harris_conductivity_dependence.md
2018_Liu_finite_offset_VSEP.md
2023_Hu_spectral_ratios.md
```
