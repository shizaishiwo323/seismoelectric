# 02_recent_seismoelectric_research_experiments_simulations

本文件夹用于存放震电研究现状相关论文，尤其是实验、数值模拟、谱比分析、有限偏移距观测、岩样/砂箱/水槽实验等材料。

适合用途：

- 写引言中的研究现状；
- 支撑“震电响应受电导率、孔隙率、渗透率、饱和度、偏移距等影响”的论述；
- 写讨论部分的实验/模拟对比；
- 查找可引用的实验现象和应用背景。

当前归入这里的材料包括 Martins-Gomes、Block & Harris、Peng、Hu、Holzhauer、Zhu、Wapenaar/Schoemaker 等实验或应用研究。
