# 01_classical_theory_foundational_papers

本文件夹用于存放经典震电理论、公式来源和奠基性模型论文。

适合用途：

- 核对震电理论公式、符号和相位约定；
- 核对 fluid/porous-medium interface 边界条件；
- 核对有限偏移距 VSEP、频率-波数积分、界面电磁响应正演；
- 写方法部分和理论框架部分。

当前归入这里的材料包括 Schakel/Smeulders、Liu finite-offset VSEP、Garambois & Dietrich full-waveform、Tardif/Jouniaux 动态电动参数、Jouniaux & Zyserman review 等。
