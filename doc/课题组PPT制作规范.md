## 一、整体主题

这份手写笔记主要记录了 **论文图表 / PPT 汇报页面的格式规范**，包括：

- 图表格式：坐标轴、线宽、背景、legend、symbol、smooth 等；
- 字体规范：中英文字体、字号、加粗要求；
- PPT 配色：模板色、中性色、主题色、强调色；
- 页面版式：标题、图文结合、图像分布、动画、页码；
- 汇报风格：简洁、清晰、准确、有品位。

## 二、第 1 页内容整理（2021/05/19）

### 2.1 图格式

手写页左侧画了一个图表示意图，标注内容如下：

- 图格式。
- 图表背景：**纯白色**。
- 图外框：**不要**。
- 图例：legend。
- 坐标轴示例：
  - 纵轴：`Removal Rate (%)`；
  - 横轴：`Time (s)`。
- 坐标刻度示例：
  - 纵轴：`0.0`、`0.5`、`1.0`、`1.5`。
- 曲线和坐标轴线宽：
  - 曲线线宽：约 `2 pt`；
  - tick 线宽：`2 pt`；
  - 坐标轴 / 线宽：`2 pt`。

### 2.2 数值模拟与实验数据表达

- 数值模拟：使用 **连续线**，即 `solid line`。
- 实验数据：使用 **点**。
- 如有必要：可以使用 `broken line for visual guidance`。

### 2.3 Symbol / 数据点符号

手写页中列出了黑白符号示例：

- `symbol`：建议使用黑白符号。
- 空心符号：
  - 空心方形；
  - 空心菱形；
  - 空心三角形；
  - 空心圆形。
- 空心符号大小：`6–8 pts`。
- 填充 / 斜线符号：
  - 斜线填充方形；
  - 斜线填充菱形；
  - 斜线填充三角形；
  - 斜线填充圆形。
- 填充 / 斜线符号大小：`5–7 pts`。

### 2.4 Smooth / 平滑处理

- 英文标注：`no smooth`。
- 中文标注：可以 `smoothen`。

### 2.5 字体大小

- Word：`12 pts`。
- PPT：`16 pts` 以上。

### 2.6 中英文字体

- 英文字体：`Arial` 或 `Times New Roman`。
- 中文字体：`SimHei / 黑体加粗`。

### 2.7 PPT 配色

- PPT 中彩色图优先使用：
  - 模板色，例如蓝色、红色；
  - 黑色。
- 不要“花里胡哨”。

## 三、第 2 页内容整理（2021/05/20）

### 3.1 标题字号与标题表达

- 标题字号：`32 pts`。
- 标题字号范围：`28–42 pts`。
- 标题表达方式：
  - 动宾结构；
  - `[疑似：推荐 6]`；
  - `提出(3)`；
  - `实验发现(3)`。

> 备注：这一行手写内容为“动宾（……）……，提出(3)……，实验发现(3)……”，中间部分字迹不完全清晰。

### 3.2 图与 bullet 的关系

手写页左侧画了一个页面布局示意图：

- 图可以放在 bullet 左右。
- 图与 bullet 的位置不要过于整齐 / 呆板。
- 图中标注如 `X-1`、`Y-2` 等要让人看清楚。
- 一组标题不多于 `2` 个。
- 一页约 `1` 分钟。
- 要有页码。

### 3.3 图文结合与页面布局

- 画面与文字要结合紧密。
- `[疑似：前后一搭桥]`，不要空。
- 画面内容要合理分布。
- 页面应当：
  - 简洁；
  - 大方；
  - 美观。
- 不要重复堆放相似图像：
  - `[疑似：2 个图像，放一个；另一个仅作装饰]`。

### 3.4 表格、动画与照片

- 不要表！
- `[疑似：懒]`。
- 不要弄动画。
- 内容一下全出来。
- 照片要简洁、美。

### 3.5 最后一张 PPT 与汇报者风格

手写页左下角有一段总结：

- 最后，一张 PPT 代表了报告者的性格 / 风采。
- 要表现出：
  - 思路清晰；
  - 表达简洁 / 准确；
  - “品位”到位。

### 3.6 汇报文字风格

右上方手写内容：

- 极简。
- 不超过 `2` 行。

### 3.7 PPT 整体配色

- PPT 整体颜色不要“跳”。
- 多用中性色：
  - “黑白灰”；
  - `[疑似：水墨色]`；
  - `[疑似：同类、冲突等]`。
- 三色原则：
  - 主题色：红；
  - 主题色：蓝；
  - 黑色。

### 3.8 黑色、强调色与配色

- 黑色：
  - `neutral`；
  - 中性；
  - 降低突兀感；
  - 用于不需要强调的内容；
  - 可用于图中的参考项、图例或色轴名等 `[疑似]`。
- 橙 / 蓝：
  - 一般用于强化；
  - 需要读出来的内容可以强化。
- 红色：
  - 用于强调的“点”；
  - 不要太多。
- 可参考：
  - 答辩贴图 / 名画 `[疑似]`；
  - 配色如何搭配；
  - 画面设计。

### 3.9 PPT 最小字号与字体

- PPT 最小字体：`16 pts`。
- 中文：
  - 黑体加粗；
  - 黑色文字可以不加粗 `[疑似]`。
- 英文：
  - 图、字：`Arial`；
  - 正文 / 文本：`Times New Roman` `[疑似]`。
