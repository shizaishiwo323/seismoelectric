# Formula Description for Reactive-Transport-Controlled Interface Seismoelectric Modeling

This document describes the formula chain implemented in
`seismoelectric_offset_liu2018_spectral.py` and
`liu2018_fig2b_dipole_comparison.py`. The model maps the evolution of pore
structure and fluid chemistry from a reactive transport simulation to
frequency-dependent electrokinetic properties, solves the fluid-porous-medium
interface conversion problem, synthesizes finite-offset electrical-potential
waveforms, and interprets their spatial amplitude pattern with a vertical
electric dipole.

The full evidence chain is

$$
\{\phi,k_0,\alpha_\infty,c_H\}(t_d)
\rightarrow
\{k(\omega),L(\omega),\sigma(\omega),\bar{\varepsilon}(\omega)\}(t_d)
\rightarrow
\{R_E,T_{TM}\}(k_x,\omega;t_d)
\rightarrow
\{R_u,T_u\}(k_x,\omega;t_d)
\rightarrow
u(x,z,t;t_d)
\rightarrow
\{p(z),|p(z)|\}
\leftrightarrow
\frac{\cos\theta}{r^2}.
$$

Here, $t_d$ is the reactive transport or dissolution time, whereas $t$ is the
wave-propagation time after acoustic excitation. These two time variables have
different physical meanings and must not be mixed.

## Formula Audit

The implementation and this document were checked against page images of Liu
et al. (2018), Eq. (1)-(4), and Schakel and Smeulders (2010), Eq. (8)-(13),
Eq. (20), Eq. (24)-(30), Eq. (36)-(46), and Appendices A-B.

The main correction to the previous document is the transition angular
frequency. Schakel Eq. (A2) contains porosity:

$$
\omega_t=
\frac{\phi\eta}{\alpha_\infty k_0\rho_f}.
$$

The two scripts otherwise follow the intended paper-level formula logic. The
most important interpretation point is that Liu Eq. (1)-(2) integrate
electrical potentials, while the Schakel boundary system returns vector-
potential coefficients. The conversion between these representations is an
implementation-level derivation from the Schakel field definitions, not an
additional empirical scaling.

| Item | Code location | Paper basis | Audit result |
| --- | --- | --- | --- |
| Dynamic permeability and transition frequency | `dynamic_coefficients()` | Schakel Eq. (A1)-(A3) | Aligned; $\omega_t$ includes $\phi$ |
| Electrokinetic coupling and conductivity | `dynamic_coefficients()` | Schakel Eq. (A4)-(A11) | Aligned |
| Effective permittivity, slownesses, and amplitude ratios | `wave_slownesses()` | Schakel Eq. (20), Eq. (24)-(30), Eq. (36)-(39) | Aligned |
| Six-equation interface problem | `se_coefficients()` | Schakel Eq. (46), Appendix B | Aligned |
| Finite-width source and spectral synthesis | `synthesize_waveforms_spectral()` | Liu Eq. (1)-(2) | Aligned, with a finite numerical spectrum |
| Pride-theory and electric-dipole comparison | `liu2018_fig2b_dipole_comparison.py` | Liu Eq. (3)-(4), Fig. 2(b) | Aligned; dipole is explanatory only |

The equations are aligned, but the project does not copy every parameter value
from Schakel Table I. The solid and frame properties are configured for the
calcite-reactive system, and $\phi$, $k_0$, $\alpha_\infty$, and $c_H$ evolve
with dissolution.

## 1. Model Logic and Notation

The model has four connected levels.

1. Reactive transport outputs are mapped to dynamic electrokinetic properties.
2. The Schakel and Smeulders boundary-value problem is solved for each
   frequency and horizontal wavenumber.
3. Schakel vector-potential coefficients are converted to the electrical-
   potential coefficients required by Liu Eq. (1)-(2), and waveforms are
   synthesized by frequency-wavenumber integration.
4. The resulting full Pride-theory waveform amplitudes are compared with the
   independent electric-dipole model in Liu Eq. (4).

Two uses of $k_0$ must be distinguished. In Schakel and Smeulders, $k_0$ is
static permeability. In Liu et al., the source spectrum uses $k_0$ for the
central horizontal wavenumber. This document writes the latter as $k_c$.

| Symbol | Meaning | Unit | Code name |
| --- | --- | --- | --- |
| $k_0(t_d)$ | Static permeability | $\mathrm{m^2}$ | `k0_m2` |
| $k_c(\omega)$ | Central horizontal source wavenumber | $\mathrm{m^{-1}}$ | `k_center` |
| $k_b$ | Source wavenumber bandwidth | $\mathrm{m^{-1}}$ | `source_kb_m_inv` |
| $k_x$ or $k$ | Horizontal integration wavenumber | $\mathrm{m^{-1}}$ | `kx` |
| $R_E$ | Schakel reflected EM vector-potential coefficient | code-scaled | `R_E` |
| $T_{TM}$ | Schakel transmitted solid TM vector-potential coefficient | code-scaled | `T_TM` |
| $R_u,T_u$ | Electrical-potential coefficients used in the Liu integral | code-scaled | returned by `liu_electrical_potential_coefficients()` |

## 2. Reactive Transport to Electrokinetic Properties

### 2.1 Inputs, Units, and Electrochemistry

At each dissolution time $t_d$, the reactive transport result provides

$$
\phi(t_d),\quad
k_0(t_d),\quad
\alpha_\infty(t_d),\quad
c_H(t_d),
$$

where $\phi$ is porosity, $k_0$ is static permeability, $\alpha_\infty$ is
high-frequency tortuosity, and $c_H$ is outlet hydrogen concentration.

Permeability is converted from mD to SI units as

$$
k_0[\mathrm{m^2}]
=
k_0[\mathrm{mD}]
\times 9.869233\times10^{-16}.
$$

When the hydrogen concentration is supplied in $\mathrm{mol/cm^3}$,

$$
c_H[\mathrm{mol/L}]
=
1000\,c_H[\mathrm{mol/cm^3}].
$$

The code applies a neutral-water floor before calculating pH:

$$
c_H^{\ast}
=
\max(c_H,10^{-7}\ \mathrm{mol/L}),
\qquad
pH=-\log_{10}c_H^{\ast}.
$$

If a measured electrolyte concentration is available, the code uses it
directly. Otherwise, it applies the first-order assumption

$$
C(t_d)=C_{\mathrm{bg}}+c_H^{\ast}(t_d).
$$

This relation is a model assumption for incomplete chemical input, not a
formula from Schakel and Smeulders.

### 2.2 Dynamic Permeability

Schakel Appendix A describes the transition from viscosity-dominated flow to
inertia-dominated flow with

$$
\omega_t(t_d)
=
\frac{\phi(t_d)\eta}
{\alpha_\infty(t_d)k_0(t_d)\rho_f},
$$

$$
k(\omega;t_d)
=
k_0(t_d)
\left[
\sqrt{
1+i\frac{\omega}{\omega_t(t_d)}\frac{M}{2}
}
+i\frac{\omega}{\omega_t(t_d)}
\right]^{-1},
$$

and

$$
M=
\frac{8\alpha_\infty k_0}{\phi\Lambda^2}.
$$

The script uses $M=1$ by default and therefore computes

$$
\Lambda(t_d)
=
\sqrt{
\frac{8\alpha_\infty(t_d)k_0(t_d)}
{\phi(t_d)M}
}.
$$

### 2.3 Zeta Potential, Debye Length, and Coupling

The semi-empirical zeta-potential relation is

$$
\zeta(t_d)
=
\left[
0.010+0.025\log_{10}C(t_d)
\right]
\frac{pH(t_d)-2}{5}.
$$

The Debye length is

$$
d(t_d)
=
\left[
\sum_l
\frac{(e z_l)^2N_l(t_d)}
{\varepsilon_0\varepsilon_f k_BT}
\right]^{-1/2}.
$$

For the binary symmetric electrolyte used in the code,

$$
z_1=1,\quad z_2=-1,\quad
N_1=N_2=N_A\,1000\,C(t_d).
$$

The electrokinetic coupling coefficient is Schakel Eq. (A4):

$$
L(\omega;t_d)
=
-
\frac{\phi(t_d)}{\alpha_\infty(t_d)}
\frac{\varepsilon_0\varepsilon_f\zeta(t_d)}{\eta}
\left(1-2\frac{d(t_d)}{\Lambda(t_d)}\right)
$$

$$
\times
\left[
1+
2i\frac{\omega}{M\omega_t(t_d)}
\left(1-2\frac{d(t_d)}{\Lambda(t_d)}\right)^2
\left(
1+d(t_d)\sqrt{\frac{i\omega\rho_f}{\eta}}
\right)^2
\right]^{-1/2}.
$$

### 2.4 Conductivity and Effective Permittivity

The pore-fluid conductivity is

$$
\sigma_f(t_d)
=
\sum_l(ez_l)^2b_lN_l(t_d).
$$

If a measured fluid conductivity is present in the input table, the code uses
it as an override. The dynamic bulk conductivity is

$$
\sigma(\omega;t_d)
=
\frac{\phi(t_d)\sigma_f(t_d)}{\alpha_\infty(t_d)}
\left[
1+
\frac{
2\{C_{\mathrm{em}}(t_d)+C_{\mathrm{os}}(\omega;t_d)\}
}
{\sigma_f(t_d)\Lambda(t_d)}
\right],
$$

where $C_{\mathrm{em}}$ and $C_{\mathrm{os}}$ are the electromigration and
electroosmotic excess conductances from Schakel Eq. (A8)-(A10).

The bulk electrical permittivity is

$$
\varepsilon(t_d)
=
\varepsilon_0
\left[
\frac{
\phi(t_d)\{\varepsilon_f-\varepsilon_s\}
}
{\alpha_\infty(t_d)}
+\varepsilon_s
\right].
$$

Schakel Eq. (20) gives the effective electrical permittivity:

$$
\bar{\varepsilon}(\omega;t_d)
=
\varepsilon(t_d)
-i\frac{\sigma(\omega;t_d)}{\omega}
+i\frac{\eta L^2(\omega;t_d)}
{\omega k(\omega;t_d)}.
$$

## 3. Schakel and Smeulders Interface Conversion

### 3.1 Biot-Willis Elastic Coefficients and Effective Densities

Let $K_s$ be the grain bulk modulus, $K_f$ the pore-fluid bulk modulus, $K_b$
the drained frame bulk modulus, and $G$ the shear modulus. Define

$$
D_K
=
K_f\left(1-\phi-\frac{K_b}{K_s}\right)+\phi K_s,
$$

$$
A
=
\frac{
(1-\phi)^2K_sK_f-(1-\phi)K_bK_f+\phi K_sK_b
}
{D_K}
-\frac{2G}{3},
$$

$$
Q
=
\frac{
\phi\{K_s(1-\phi)-K_b\}K_f
}
{D_K},
\qquad
R
=
\frac{\phi^2K_sK_f}{D_K},
\qquad
P=A+2G.
$$

The frequency-dependent density terms are

$$
\rho_{12}(\omega;t_d)
=
\phi\rho_f
\left[
1+i\frac{\phi\eta}
{\omega\rho_fk(\omega;t_d)}
\right],
$$

$$
\rho_{11}
=(1-\phi)\rho_s-\rho_{12},
\qquad
\rho_{22}
=\phi\rho_f-\rho_{12}.
$$

Electrokinetic coupling introduces

$$
E_K(\omega;t_d)
=
\frac{
\eta^2\phi^2L^2(\omega;t_d)
}
{
k^2(\omega;t_d)\bar{\varepsilon}(\omega;t_d)\omega^2
},
$$

$$
\bar{\rho}_{11}=\rho_{11}-E_K,\qquad
\bar{\rho}_{12}=\rho_{12}+E_K,\qquad
\bar{\rho}_{22}=\rho_{22}-E_K.
$$

### 3.2 Complex Slownesses and Amplitude Ratios

For the fast and slow longitudinal modes,

$$
d_0
=
\bar{\rho}_{11}\bar{\rho}_{22}-\bar{\rho}_{12}^2,
$$

$$
d_1
=
-
\left(
P\bar{\rho}_{22}
+R\bar{\rho}_{11}
-2Q\bar{\rho}_{12}
\right),
\qquad
d_2=PR-Q^2.
$$

The squared complex slownesses are

$$
s_l^2
=
\frac{-d_1/d_2
\pm
\sqrt{(d_1/d_2)^2-4d_0/d_2}}
{2},
\qquad
l=Pf,Ps.
$$

For the transverse EM and SV modes,

$$
d_{0T}
=
\frac{
\mu\bar{\varepsilon}
\left(
\bar{\rho}_{11}\bar{\rho}_{22}-\bar{\rho}_{12}^2
\right)
}
{G},
$$

$$
d_{1T}
=
-\mu\bar{\varepsilon}\bar{\rho}_{22}
-\frac{\rho_{11}\rho_{22}-\rho_{12}^2}{G},
\qquad
d_{2T}=\rho_{22}.
$$

The same quadratic expression gives $s_{TM}^2$ and $s_{SV}^2$. The code sorts
the roots by magnitude, assigning the smaller-magnitude roots to the fast
longitudinal and EM-dominated modes.

For horizontal wavenumber $k_1=k_x$, each vertical wavenumber is

$$
k_3^m
=
\sqrt{\omega^2s_m^2-k_1^2}.
$$

For the reflected EM mode in the upper fluid, the code uses

$$
s_E^2
=
\mu_0\varepsilon_0\varepsilon_{fl}
-i\frac{\mu_0\sigma_{fl}}{\omega},
\qquad
k_3^E=\sqrt{\omega^2s_E^2-k_1^2}.
$$

The default upper-fluid conductivity is the constant value
$\sigma_{fl}=5.0\times10^{-3}\ \mathrm{S/m}$ from Schakel Table I. An optional
code mode can instead use the pore-fluid conductivity estimate.

The code follows the Schakel convention

$$
\exp(i\omega t)\exp(-i\mathbf{k}\cdot\mathbf{x})
$$

and selects square-root branches that satisfy the radiation and decay
conditions.

The longitudinal amplitude ratios are

$$
\beta_m
=
\frac{\bar{\rho}_{11}-Ps_m^2}
{Qs_m^2-\bar{\rho}_{12}},
$$

$$
\alpha_m
=
\frac{\eta\phi L}
{k(\omega)\bar{\varepsilon}}
\left(1-\beta_m\right),
\qquad
m=Pf,Ps.
$$

The transverse amplitude ratios are

$$
\beta_n
=
\frac{Gs_n^2-(1-\phi)\rho_s}
{\phi\rho_f},
$$

$$
\alpha_n
=
\frac{\mu\eta\phi L}
{k(\omega)\{\mu\bar{\varepsilon}-s_n^2\}}
\left(1-\beta_n\right),
\qquad
n=TM,SV.
$$

### 3.3 Six-Equation Boundary-Value Problem

The Schakel unknown vector is

$$
\mathbf{x}
=
\left[
R_E,\ R_M,\ T_{Pf},\ T_{Ps},\ T_{TM},\ T_{SV}
\right]^T.
$$

In the paper notation, these are the coefficients
$[R^E,R^M,T^{Pf},T^{Ps},T^{TM},T^{SV}]^T$ in Eq. (46). The code solves

$$
\mathbf{A}(\omega,k_x;t_d)\mathbf{x}=\mathbf{b}
$$

with the Appendix B matrix elements for each $(k_x,\omega)$ pair. Thus,
$R_E$ and $T_{TM}$ are not empirical conversion factors.

Schakel Eq. (39) gives the electric vector-potential amplitude associated with
the transmitted solid TM vector potential:

$$
T_E^{(\Psi)}
=
\alpha_{TM}T_{TM}.
$$

At exact normal incidence, the raw Schakel coefficients $R_E$ and $T_{TM}$
vanish under the prescribed boundary conditions. This is why the
dissolution-time coefficient curves use a representative incidence angle of
$45^\circ$.

### 3.4 Conversion from Schakel Vector Potentials to Liu Electrical Potentials

Liu Eq. (1)-(2) uses electrical potentials in the fluid and porous sandstone.
Schakel Eq. (23), however, writes the electric field as

$$
\mathbf{E}
=
\nabla\phi_E+\nabla\times\mathbf{\Psi}_E.
$$

For a two-dimensional TM vector potential

$$
\mathbf{\Psi}_E=(0,\psi_E,0),
$$

the horizontal electric field is

$$
E_x=-\frac{\partial\psi_E}{\partial z}.
$$

Relating this field to a scalar electrical potential through
$E_x=-\partial u/\partial x$, and accounting for the opposite reflected and
transmitted vertical propagation directions in Schakel Eq. (42) and Eq. (44),
gives the code-level Liu coefficients

$$
R_u(|k_x|,\omega)
=
-
\frac{k_3^E}{|k_x|}
R_E(|k_x|,\omega),
$$

$$
T_u(|k_x|,\omega)
=
+
\frac{k_3^{TM}}{|k_x|}
\alpha_{TM}(|k_x|,\omega)
T_{TM}(|k_x|,\omega).
$$

The opposite signs follow from the field representation and propagation
directions. They are not empirical polarity corrections.

The signed $k_x$ remains in the Fourier phase, while the isotropic interface
coefficients are evaluated as functions of incidence-angle magnitude,
$|k_x|$. At the central $k_x=0$ quadrature node, the code evaluates the finite
potential-coefficient limit at a very small positive wavenumber, while the
source spectrum and lateral Fourier phase retain the true value $k_x=0$.

## 4. Liu-Style Frequency-Wavenumber Waveform Synthesis

### 4.1 Finite-Width Acoustic Source

Liu et al. represent the finite-width source by

$$
A(k_x,\omega)
=
p_0S(\omega)
\exp\left[
-\frac{(k_x-k_c)^2}{k_b^2}
\right],
$$

where $p_0$ is an arbitrary source amplitude, $S(\omega)$ is the source
spectrum, $k_b$ controls beam width, and

$$
k_c(\omega)
=
\frac{\omega}{V_f}\sin\theta_b,
\qquad
V_f=\sqrt{\frac{K_{fl}}{\rho_{fl}}}.
$$

The default $\theta_b=0^\circ$ gives a vertically centered acoustic beam.
Because $k_b$ is finite, the source still contains nonzero horizontal
wavenumbers that can contribute to interface conversion.

The default geometry and source settings follow the Liu finite-offset example:
$z_s=80\ \mathrm{mm}$, $D=45\ \mathrm{mm}$, $k_b=80\ \mathrm{m^{-1}}$, and
$f_0=500\ \mathrm{kHz}$.

### 4.2 Causal Ricker Source Spectrum

The Schakel and Liu formulas use the $\exp(i\omega t)$ time convention. The
code therefore uses

$$
S(\omega)
=
\int s(\tau)\exp(-i\omega\tau)\,d\tau,
\qquad
s(t)
=
\frac{1}{2\pi}
\int S(\omega)\exp(i\omega t)\,d\omega.
$$

The source-time Ricker wavelet is

$$
s_R(\tau)
=
\left[
1-2\{\pi f_0(\tau-\tau_p)\}^2
\right]
\exp\left[
-\{\pi f_0(\tau-\tau_p)\}^2
\right].
$$

The script applies a smooth onset ramp, integrates the source over a finite
duration, and normalizes the resulting spectrum by its maximum magnitude.
This places the source delay in the frequency-domain phase instead of applying
a receiver-side time gate.

### 4.3 Reflected and Transmitted Electrical Potentials

For a receiver in the upper fluid, Liu Eq. (1) is implemented as

$$
u_r(x,z,t;t_d)
=
\int_{\omega_{\min}}^{\omega_{\max}}
\int_{-k_{\max}}^{k_{\max}}
A(k_x,\omega)
R_u(|k_x|,\omega;t_d)
$$

$$
\times
\exp\left\{
-i\left[
k_3^iz_s-k_3^Ez+k_xx-\omega t
\right]
\right\}
\,dk_x\,d\omega.
$$

For a receiver in the porous medium, Liu Eq. (2) is implemented as

$$
u_t(x,z,t;t_d)
=
\int_{\omega_{\min}}^{\omega_{\max}}
\int_{-k_{\max}}^{k_{\max}}
A(k_x,\omega)
T_u(|k_x|,\omega;t_d)
$$

$$
\times
\exp\left\{
-i\left[
k_3^iz_s+k_3^{TM}z+k_xx-\omega t
\right]
\right\}
\,dk_x\,d\omega.
$$

The lateral receiver offset is $x=D$. The signed $k_x$ controls the Fourier
phase $\exp(-ik_xD)$, while the interface coefficients depend on $|k_x|$.

### 4.4 Numerical Spectral Window

The code uses a finite positive-frequency window,

$$
\omega\in
\left[
2\pi c_{\min}f_0,\,
2\pi c_{\max}f_0
\right],
$$

with default values $c_{\min}=0.25$ and $c_{\max}=2.5$. The horizontal
wavenumber range is limited to propagating incident-fluid components:

$$
|k_x|
\le
\gamma\frac{\omega}{V_f},
\qquad
\gamma=0.98.
$$

Only positive frequencies are explicitly integrated. The factor of two in the
quadrature weight accounts for the complex-conjugate negative-frequency
contribution required for a real time-domain waveform:

$$
u(x,z,t;t_d)
\approx
\sum_j\sum_l
2A(k_l,\omega_j)
C(|k_l|,\omega_j;t_d)
\exp[-i\Phi(k_l,\omega_j;x,z,t)]
w_{\omega,j}w_{k,l}(2\pi)^{-2},
$$

where $C=R_u$ or $T_u$.

## 5. Electric-Dipole Interpretation of the Pride-Theory Prototype

### 5.1 Meaning of "Pride Theory" in the Comparison Script

In `liu2018_fig2b_dipole_comparison.py`, "Pride theory" denotes the complete
electrical-potential waveform generated by
`seismoelectric_offset_liu2018_spectral.py`. Pride/Schakel electrokinetic
coupling determines the interface coefficients, and Liu Eq. (1)-(2)
determines the finite-width, finite-offset spectral waveform.

The electric-dipole model is an independent explanation curve from Liu
Eq. (3)-(4). It is not multiplied into the Pride-theory waveform.

### 5.2 Receiver-Line Geometry and Dipole Potential

For a receiver line at horizontal offset $D$,

$$
x=D,\qquad z=z_r,
$$

$$
r=\sqrt{D^2+z_r^2},
\qquad
\cos\theta=\frac{z_r}{r}.
$$

Liu Eq. (3) writes the vertical dipole potential as

$$
u
=
f(\mathbf{P},\varepsilon)
\frac{\cos\theta}{r^2}.
$$

For a fixed medium, $f(\mathbf{P},\varepsilon)$ is treated as a constant, so
Liu Eq. (4) becomes

$$
u
\propto
\frac{\cos\theta}{r^2}.
$$

The comparison script therefore evaluates

$$
g_D(z_r)
=
\frac{\cos\theta}{r^2}
=
\frac{z_r}{(D^2+z_r^2)^{3/2}}.
$$

Along a fixed-offset receiver line, $r=D/\sin\theta$, so the normalized dipole
amplitude has the angular form

$$
|g(\theta)|
\propto
|\cos\theta|\sin^2\theta.
$$

The maximum amplitude occurs at

$$
|z_r|=\frac{D}{\sqrt{2}},
$$

which corresponds to approximately $54.7^\circ$ and $125.3^\circ$ on the two
sides of the interface.

### 5.3 Exact Comparison Performed by `liu2018_fig2b_dipole_comparison.py`

For each receiver position $z_j$, the script extracts the signed waveform
sample with the largest absolute magnitude:

$$
p_j
=
U\!\left(
z_j,
\operatorname*{arg\,max}_t|U(z_j,t)|
\right),
\qquad
a_j=|p_j|.
$$

It then computes

$$
g_j
=
\frac{z_j}{(D^2+z_j^2)^{3/2}}
$$

independently of the waveform. The Pride-theory peak amplitudes and the
electric-dipole amplitudes are normalized and plotted against radiation angle.
The reflected and transmitted Pride-theory amplitudes are normalized
separately because their absolute coefficient scales differ. This
normalization is used to compare spatial shape, not absolute amplitude across
the interface.

The dipole model explains two features of the full spectral result:

1. The sign change of $\cos\theta$ explains the polarity reversal across the
   interface.
2. The combination of angular directivity and $r^{-2}$ decay explains the
   non-monotonic finite-offset amplitude pattern.

The slightly broader Pride-theory pattern can be attributed to the finite
acoustic beam represented by $A(k_x,\omega)$, consistent with the
interpretation given by Liu et al. for Fig. 2(b).

## 6. Output Quantities

### 6.1 Dissolution-Time Coefficient Curves

The coefficient curves use

$$
\omega_0=2\pi f_0,
\qquad
\theta_0=45^\circ,
$$

and report

$$
|R_E(\omega_0,\theta_0;t_d)|,
\qquad
|T_{TM}(\omega_0,\theta_0;t_d)|.
$$

The normalized monitoring curves are

$$
\frac{|R_E(t_d)|}{|R_E(t_{d,0})|},
\qquad
\frac{|T_{TM}(t_d)|}{|T_{TM}(t_{d,0})|}.
$$

### 6.2 Waveform Snapshot

At a selected dissolution time $t_d^\ast$, the spectral synthesis gives

$$
u(z_r,t;t_d^\ast).
$$

The acoustic travel time from the source to the interface is

$$
T_0=\frac{z_s}{V_f}.
$$

The reflected and transmitted waveform sides are normalized separately for
display. This makes both waveform families visible but must not be interpreted
as an absolute amplitude comparison.

### 6.3 Maximum Waveform Amplitude

For each dissolution time,

$$
A_{\max}(t_d)
=
\max_{z_r,t}|u(z_r,t;t_d)|.
$$

If signed extrema are needed,

$$
A_{\mathrm{peak}}(t_d)
=
\max_{z_r,t}u(z_r,t;t_d),
$$

$$
A_{\mathrm{trough}}(t_d)
=
\min_{z_r,t}u(z_r,t;t_d).
$$

## 7. Applicability and Paper-Writing Logic

### 7.1 Poroelastic Validity Range

The Biot-type model requires a load-bearing solid frame. The script uses

$$
\phi_{\max}=0.95
$$

as the upper validity limit. Results above this value should not be interpreted
as a physical seismoelectric trend because the assumed porous frame is nearly
absent.

### 7.2 Method Paragraph

> At each dissolution time, porosity, permeability, tortuosity, and outlet hydrogen concentration from the reactive transport simulation were mapped to the frequency-dependent electrokinetic properties of the porous medium. Dynamic permeability, electrokinetic coupling, and dynamic conductivity were computed following the Schakel and Smeulders formulation. The reflected EM vector-potential coefficient and the transmitted solid TM vector-potential coefficient were obtained by solving the fluid-porous-medium boundary-value problem for each frequency and horizontal wavenumber. These Schakel potentials were converted to the electrical-potential coefficients used in the Liu et al. formulation before the interface EM waveforms were synthesized with a finite-width acoustic source and a finite-offset frequency-wavenumber integral. The spatial amplitude pattern of the complete Pride-theory waveform was then interpreted with the vertical electric-dipole radiation factor.

### 7.3 Evidence Function of Each Model Level

| Model level | Scientific question | Evidence produced |
| --- | --- | --- |
| Reactive transport mapping | How does dissolution change pore structure and fluid chemistry? | $\phi,k_0,\alpha_\infty,pH,C$ |
| Electrokinetic properties | How do structural and chemical changes modify coupling? | $k(\omega),L(\omega),\sigma(\omega)$ |
| Schakel boundary problem | How does the interface convert an incident acoustic wave to EM modes? | $R_E,T_{TM}$ |
| Schakel-to-Liu conversion | How do vector potentials map to receiver electrical potentials? | $R_u,T_u$ |
| Liu spectral synthesis | How do source width and receiver offset form the waveform? | $u(x,z,t;t_d)$ |
| Electric-dipole interpretation | Why does the full Pride-theory amplitude vary with receiver position? | $\cos\theta/r^2$ |
| Monitoring metrics | How does dissolution appear in an observable signal? | $A_{\max}(t_d)$ and normalized coefficients |

This structure keeps the method centered on a mechanism chain: reactive
transport changes medium properties, medium properties change electrokinetic
coupling, coupling changes interface conversion, and the conversion
coefficients combine with source and receiver geometry to determine the
observed electrical-potential waveform.
