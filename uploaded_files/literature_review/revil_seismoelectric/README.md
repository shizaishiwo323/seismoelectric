# Revil seismoelectric literature

This folder is the canonical local collection for Andre Revil's seismoelectric, electroseismic, and electrokinetic-coupling papers/books used by this project.

Scope:

- Included: Revil-authored or Revil-edited works directly relevant to seismoelectric theory, electrokinetic coupling, two-phase/unsaturated porous media, seismic focusing, cross-hole imaging, and joint inversion.
- Excluded: general seismic full waveform inversion papers without Revil authorship or direct seismoelectric coupling content.

## Fast Orientation

For this project, read in this order:

1. `2015_Revil_Jardani_Sava_Haas_The_Seismoelectric_Method_book.pdf`
   - Best broad reference for Revil's seismoelectric framework, terminology, applications, and modeling philosophy.
   - Use as a background/theory map, not as the sole formula source for the current Schakel/Liu implementation.

2. `2007_Revil_Linde_Cerepi_electrokinetic_coupling_unsaturated_porous_media.pdf`
   - Key paper for effective excess charge density, saturation-dependent electrokinetic coupling, and the alternative charge-density parameterization.
   - Useful when discussing how pore structure, saturation, salinity, and hydraulic state control electrokinetic source strength.

3. `2013_Revil_Mahardika_coupled_hydromechanical_electromagnetic_unsaturated.pdf`
   - Extends coupled hydromechanical/electromagnetic disturbances to unsaturated porous media.
   - Important for interpreting water/gas/oil saturation fronts as moving electrokinetic sources.

4. `2014_Revil_Barnier_Karaoulis_Sava_unsaturated_porous_media.pdf`
   - Connects seismoelectric coupling, petrophysical parameters, and saturation-front localization using an electroacoustic approach.
   - Most relevant Revil paper for linking heterogeneous hydrologic fronts to remotely measurable seismoelectric signals.

5. `2015_Jardani_Revil_two_immiscible_fluid_phases.pdf`
   - Formal two-immiscible-fluid seismoelectric coupling model.
   - Useful for oil-water or water-air analogies, and for discussing saturation-front physics beyond fully saturated Pride/Schakel theory.

## Relationship to Current Schakel/Liu Model

The current project model mainly follows Pride/Schakel/Smeulders-style dynamic electrokinetic coefficients and Liu-style frequency-wavenumber finite-offset waveform synthesis. Revil's papers are most useful for:

- explaining electrokinetic source physics from the electrical double layer and effective excess charge density;
- supporting discussions of saturation, fluid-property, salinity, and permeability controls on coupling strength;
- motivating moving-interface or moving-front interpretations;
- comparing interface-generated fields, cross-hole imaging, and beamforming ideas with this project's finite-offset interface EM response.

Do not directly substitute Revil's effective-charge-density formulation into the current Schakel/Liu forward code unless the governing equations and parameter mapping are explicitly revised. In the present codebase, Revil literature should be treated mainly as theoretical support, interpretation context, and possible future-model extension.
